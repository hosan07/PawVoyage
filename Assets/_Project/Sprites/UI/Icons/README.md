# Paw Voyage UI Icons

This folder contains the UI icon source set for the current mockup and future Unity UI.

## Naming

- `icon_nav_*`: five-tab navigation and top navigation icons.
- `icon_currency_*`: persistent currency indicators.
- `icon_action_*`: controls and primary actions.
- `icon_status_*`: combat and state feedback.
- `icon_system_*`: login and settings rows.
- `icon_farm_*`: farm buildings and permanent farm upgrades.
- `icon_farmer_*`: farmer portrait and permanent farmer upgrades.
- `icon_companion_*`: pet collection, pet selection, and pet tokens.

## Usage rules

- Use the same filename across mockup, Unity prefab, and design documentation.
- `icon_currency_coin.svg` is only for Farm Coin, `icon_currency_gem.svg` only for cosmetics, and `icon_currency_energy.svg` only for entry energy.
- The center deploy tab always uses `icon_nav_deploy.svg`.
- Keep the farmer portrait on every management and battle token surface as `icon_farmer_avatar.svg`.
- Use the facility and upgrade icons below without renaming them: barn, fence, pen, lantern, tool, clothes, boots, and magnet.
- Keep game characters, pets, enemies, and buildings in their own art folders. They are not UI icons.
- These are SVG source assets for the prototype. Before release, export the approved final art at 2x PNG and preserve the filenames.
